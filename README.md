# Sistema de Controle de Telescopio Espacial Compartilhado

Sistema desenvolvido para gerenciar o agendamento de um telescópio espacial acadêmico de alto custo. Devido à alta demanda global por tempo de observação, o sistema não pode apenas registrar agendamentos; ele precisa garantir justiça, consistência e exclusão mútua em um ambiente de alta concorrência .

O objetivo central é impedir que dois cientistas reservem o mesmo horário simultaneamente e garantir que o horário registrado seja preciso, independentemente do relógio local do usuário.**
